# Изображения проектов

Положите картинки для каждого проекта в соответствующую папку:

- **evaryst/** — изображение для проекта Evaryst  
  Файл: `cover.png` (или `cover.jpg`)

- **travellizy/** — изображение для проекта Travellizy  
  Файл: `cover.png` (или `cover.jpg`)

- **carforce/** — изображение для проекта Carforce  
  Файл: `cover.png` (или `cover.jpg`)

Если вы используете другое имя или расширение, напишите — изменю пути в коде.
